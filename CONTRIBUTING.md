# Contributing

Thanks for your interest in contributing!

## Guidelines

- **Look at existing packages for guidance.** Before adding or modifying a package, browse `pkgs/` to see how similar packages are structured. Follow the conventions already in place.
- **Using AI is totally acceptable.** Feel free to use Claude, Copilot, or any other tool. The only ask is to keep issue and PR titles and descriptions short and to the point. No walls of text, no lengthy AI-generated preambles.
- **No hard restrictions.** There are no strict rules or required formats beyond the above. Reasonable contributions are welcome.

If you are unsure about something, open an issue or PR and we can discuss it.
